Opening web page

Interface declaration

{"name": "system.webview"}

Import module

import webview from '@system.webview' or var webview = require("@system.webview")

WebView internal API

APIs that can be used in web pages opened in WebView

system.go(path)

Return specified page of the current app

Parameters:

Parameter nameTypeRequiredDescription
pathStringYesThe page to return to, e.g.: /detail?param1=value1, if the value of the path is "/", then the path jumped to is the "/" page, if there is none, then it will jump to the home page

Example:

system.go("/detail?param1=value1")
